# Spellbook 魔法书品牌与产品语义

## 品牌

- 中文名：`魔法书`
- 英文名：`Spellbook`
- 完整品牌名：`Spellbook 魔法书`
- 副标题：`AI 提示词与技能管理器`
- 一句话定位：把可复制的提示词文本和可迁移的 Agent 技能包整理到一个本地魔法书里。

## 核心资产

### 咒语 Spell

咒语是一段用户会直接粘贴给模型的原始文本。它可以包含 `#`、列表、代码块等 Markdown 符号，但产品只把它当作纯文本 `body` 处理。

第一版规则：

- 复制时只复制 `body` 原文。
- 不自动添加标题、slug、frontmatter 或包装说明。
- 不提供文件导出。
- 从历史扫描中生成的候选项叫 `候选咒语 / Candidate Spells`。

### 技能 Skill

技能是 Claude/Codex 可安装调用的本地多文件能力包。它可以包含 `SKILL.md`、脚本、模板和资源文件。

第一版规则：

- 技能库只负责扫描、查看、打包和安装。
- 打包时保留目录内所有文件和相对路径。
- 安装到目标平台时复制整个目录。
- 目标已存在时不覆盖，返回冲突提示。

## 来源 Source

来源描述资产从哪里来，不等于资产类型。

- Claude 历史记录
- Codex 历史记录
- Claude 技能目录：`~/.claude/skills`
- Codex 用户级技能目录：`$HOME/.agents/skills`

Codex repo 级技能目录 `.agents/skills` 后续作为路径配置扩展，不作为第一版默认扫描范围。

## 命名规范

| 中文 | English | 用途 |
| --- | --- | --- |
| 魔法书 | Spellbook | 产品名 |
| 咒语库 | Spell Library | 纯文本咒语管理 |
| 技能库 | Skill Library | 多文件 Agent Skill 管理 |
| 候选咒语 | Candidate Spells | 历史扫描生成的待保存文本 |
| 快捷面板 | Quick Panel | 快速搜索并复制咒语 |

## UI 风格

Spellbook 的界面仍是克制的工具型桌面应用。魔法元素只作为轻量品牌信号：

- 品牌 mark 使用书本/星点感图标。
- 使用低饱和金色 accent。
- 保持紧凑表格、列表和设置行。
- 不使用夸张插画、渐变球、装饰堆叠。
- 功能按钮保持专业动词，例如复制、扫描、打包、安装。

## 后续概念

`Workflow`、`Memory`、`Chapter` 暂不进入第一版主导航。未来如果需要，可以作为技能或知识组织层继续扩展。

CLI 名称 `spell` 只作为后续规划保留，第一版优先完成桌面端体验。
